# 📝 NoteSphere

**NoteSphere** is a feature-rich desktop note-taking application built using **Java Swing** and **Maven**. It provides a clean and modern interface for creating, organizing, securing, and managing personal notes.

## ✨ Features

- 📝 Create and edit notes
- 🔍 Instant note search
- 🗂️ Organize notes by category
- 📌 Pin important notes
- ⭐ Mark notes as favorites
- 📷 Attach images to notes
- 🔔 Set reminder notifications
- 🗑️ Trash, restore, and permanently delete notes
- 📊 Analytics dashboard with category visualization
- 📄 Export notes to PDF
- 🔒 Password-protected application access
- 🔑 Change password functionality
- 🌙 Dark and light theme support
- 💾 Automatic and manual backups
- ♻️ Restore notes from backups
- ⌨️ Keyboard shortcuts
- 📈 Note statistics dashboard

## 🛠️ Technologies Used

- Java 17
- Java Swing
- Maven
- FlatLaf
- JFreeChart
- iText PDF
